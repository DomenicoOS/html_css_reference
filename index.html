CSS

Basics
Selector Description
* { } * is a universal selector. It can be used to select any and all types of elements in an HTML page. The asterisk can also be followed by a selector while using to select a child object. This selector is useful when we want to select all the elements on the page
Color
Property Description Values
color: Sets the color of the element by color name (there are 147 color names) blue;
Sets the color by hex code #0000FF;
Sets the color by rgb (red, green, blue) with either values 0-255 or percentages rgb(0, 0, 255); or rgb(0%, 0%, 100%);
Can also set opacity with rgba (alpha) using a value between 0 (0% opacity) and 1.0 (100% opacity) rgb(0, 0, 255, 0.5);
Sets the color by hsl (hue, saturation, lightness) with hue values between 0-360, and saturation and lightness as percentages from 0%-100% hsl(240, 100%, 50%);
Can also set opacity with hsla (alpha) using a value between 0 (0% opacity) and 1.0 (100% opacity) hsla(240, 100%, 50%, 0.5);
Box Model
Property Description Values
width: The width of the content area. Set in pixels 1600px;
min-width: Ensures a minimum width of an element’s box 300px;
max-width: Ensures a maximum width of an element’s box 600px;
height: The height of the content area. Set in pixels 800px;
min-height: Ensures a minimum height for an element’s box 150px;
max-height: Ensures a maximum height of an element’s box 300px;
padding: The amount of space between the content area and the border on all four sides 10px; or 5px(top and bottom) 10px(left and right); or 5px(top) 10px(left and right) 20px(bottom); or 5px 10px 15px 20px; (clockwise from top)
padding-top: The amount of space between the top of the content area and the top border 10px;
padding-right: The amount of space between the right of the content area and the right border 10px;
padding-bottom: The amount of space between the bottom of the content area and the bottom border 10px;
padding-left: The amount of space between the left of the content area and the left border 10px;
border: The thickness and style of the border surrounding the content area and padding. Can set width, style, and color 3px solid coral;
border-radius: Modifies the corners of an element’s border box. Can be set in pixels and percentage 5px; 50%; (perfect circle)
margin: The amount of space between the border and the outside edge of the element. Used to specify the size of the space directly outside of the box 10px; or 5px(top and bottom) 10px(left and right); or 5px(top) 10px(left and right) 20px(bottom); or 5px 10px 15px 20px; (clockwise from top)
Centers the content in their containing elements as long as a width is set. The 0 sets the top and bottom margins to 0 pixels. Auto instructs the browser to adjust the left and right margins until the element is centered within its containing element 0 auto;
margin-top: The amount of space between the top border and the top, outside edge of the element 10px;
margin-right: The amount of space between the right border and the right, outside edge of the element 10px;
margin-bottom: The amount of space between the bottom border and the bottom, outside edge of the element 10px;
margin-left: The amount of space between the left border and the left, outside edge of the element 10px;
overflow: Controls what happens to content that spills, or overflows, outside its box
Any content that overflows will be hidden from view hidden;
A scrollbar will be added to the element’s box so that the rest of the content can be viewed by scrolling scroll;
Default value. The overflow content will be displayed outside of the containing element visible;
